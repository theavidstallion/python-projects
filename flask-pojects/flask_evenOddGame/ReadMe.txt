Flask powered game of "Guess - Even or Odd"
Description:
The game is played on browser with an easy user-interface. The web app generates a random number between 1 and 10 and the user has to guess if it's even or odd.
The web app then validates the input if it's right or wrong, and then gives the result while storing the count for an instance of the web app. After that, the
user can end the game or play again.

Components of WebApp:
  Flask Libraries
  I/O
  Data Storage for an instance
  Validation
  Error Exception
  Control Flow

Heirarchy of files:
static
  styles.css
templates
  index.html
  results.html
app.py


Establish a local host/server using Flask that must be installed on your device.
And then run app.py through flask.
